## Personal site showcase

Collection of user interfaces made up with ReactJS & 👩🏻‍🎤 Emotion (CSS in JS). Most of design elements are inspired by the Refactoring UI book.

Stacks:
- NextJS
- Chakra UI
- DayJS
- React Table


Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
